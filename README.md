# HyperDrive — Luxury Supercar Showroom

A fully responsive, premium supercar showroom web app built with React and Vite.

## Features

- 🎬 Video hero carousel with per-word text animations
- 🚗 Full car inventory with brand/fuel/price filters
- 💎 Limited Edition collection page with gold accents
- 🖼️ Gallery with category filters and modal lightbox
- 📬 Contact form with real email delivery via Web3Forms
- 🔐 Login/Register page with video background
- ℹ️ About page with animated counters, timeline, and process steps
- ✨ Scroll-triggered animations throughout
- 📱 Fully responsive — mobile, tablet, desktop
- 🔤 Playfair Display + Inter typography
- ⚡ Loading screen with progress bar

## Tech Stack

- React 18
- Vite
- React Router DOM
- Web3Forms (contact form)
- CSS (no UI library — fully custom)

## Getting Started

npm install
npm run dev
